## Design Rationale

### Why These Screenshots?

The 6 screenshots were strategically selected to highlight:

1. **Dashboard** — First impression; shows comprehensive overview and professional design
2. **Job Details** — Core functionality; demonstrates job acquisition process
3. **Earnings** — Key value proposition; transparent payment structure
4. **Profile** — Trust building; shows professional verification and ratings
5. **Job Management** — Operational efficiency; demonstrates workflow tools
6. **Notifications/Map** — Real-time connectivity; shows location-based features

This sequence follows Google Play Store best practices:
- Lead with strongest visual (dashboard)
- Showcase core value proposition early (jobs, earnings)
- Build credibility (profile, ratings)
- Demonstrate complete workflow (management)
- End with unique features (real-time notifications)

### Branding Consistency

All assets maintain consistent:
- Color palette (professional blue #1E3A8A, grey #64748B)
- Typography (clear, readable fonts)
- UI patterns (rounded corners, cards, modern design)
- Messaging (professional, trustworthy, UK-focused)
