# LockSafe Store Screenshots

This folder should contain screenshots for the Google Play Store and Apple App Store.

## Required Screenshots

### Google Play Store
- **Minimum**: 2 screenshots
- **Recommended**: 8 screenshots
- **Phone Size**: 1080x1920 or 1440x2560 pixels (16:9 ratio)
- **Feature Graphic**: 1024x500 pixels

### Apple App Store
- **iPhone 6.7"**: 1290x2796 pixels
- **iPhone 6.5"**: 1242x2688 pixels
- **iPhone 5.5"**: 1242x2208 pixels
- **iPad Pro 12.9"**: 2048x2732 pixels (if supporting iPad)

## Suggested Screenshot Sequence

1. **Welcome Screen** - Show the app's value proposition
2. **Request Flow** - Show how easy it is to request help
3. **Locksmith Selection** - Show available locksmiths with ratings
4. **Live Tracking** - Show real-time locksmith tracking
5. **Quote Screen** - Show transparent pricing
6. **Digital Signature** - Show the secure completion process
7. **Job History** - Show past jobs and receipts
8. **Locksmith Dashboard** - Show locksmith features (for locksmith version)

## Screenshot Creation Tools

### Taking Screenshots
- **iOS Simulator**: Press `Cmd + S`
- **Android Emulator**: Click camera button or `Cmd + S`
- **Physical Device**: Use native screenshot (Power + Volume)

### Device Frame Tools
- [Shotsnapp](https://shotsnapp.com) - Free device frames
- [AppMockUp](https://app-mockup.com) - Marketing mockups
- [Screely](https://screely.com) - Browser mockups
- [MockUPhone](https://mockuphone.com) - Device mockups

### Screenshot Enhancement
- Add device frames for a professional look
- Add marketing text/captions above or below
- Keep text minimal - let the app speak for itself
- Use consistent styling across all screenshots
- Match the brand colors (orange #f97316)
